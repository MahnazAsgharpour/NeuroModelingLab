This repository contains Python and NEURON code to simulate the effects of Transcranial Magnetic Stimulation on neuronal cells.
# NeuroModelingLab

Computational neuroscience models and scientific software for neural modeling, extracellular stimulation, and biophysical neuron simulations.

## Features

- NEURON simulations
- Purkinje cell models
- Cable models
- Axon activation models
- Extracellular stimulation
- Python scientific workflows

## Installation

pip install -r requirements.txt
